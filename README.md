
#   微博网页端简洁化
##  使用方法
*   下载到本地解压
*   进入Chrome的“扩展程序页面”，并开启 “开发者模式”
*   点击“加载已解压的扩展程序”选择本项目
*   进入微博网页端，自动加载功能
